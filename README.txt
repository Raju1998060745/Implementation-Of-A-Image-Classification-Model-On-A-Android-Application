# Grocery Shopping App

The Grocery Shopping App is an Android application that provides a convenient and efficient way to manage your grocery shopping. It utilizes advanced machine learning techniques for accurate item classification and offers seamless integration with Firebase for data management. With the app, you can easily capture images of grocery items, identify them using the built-in image classifier, and add them to your shopping cart.

## Dataset
    Fruits 360 dataset: A dataset of images containing fruits and vegetables(https://www.kaggle.com/datasets/moltean/fruits)

## Features

- Image Classification: The app uses a pre-trained machine learning model to classify grocery items based on captured images.

- Firebase Integration: Firebase is used for seamless data management, including user authentication and cloud storage for the shopping cart.

- Real-time Processing: The TensorFlow Lite framework enables real-time image processing and classification on the user's device, reducing reliance on server-side computations.

- User-Friendly Interface: The app provides a smooth and intuitive user interface for capturing images, managing the shopping cart, and browsing item classifications.

## Installation

1. Open the project in Android Studio.

2. Build and run the app on an Android device or emulator.

## Usage

1. Launch the app on your Android device.

2. Grant the necessary permissions for camera access.

3. Tap the "Scan" button to capture an image of a grocery item.

4. Wait for the app to classify the item and display the results.

5. Choose to add the item to your shopping cart or continue scanning more items.

6. View and manage your shopping cart in the Cart section of the app.
